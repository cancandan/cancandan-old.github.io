Midi record and play in common music

Sat 02 January 2016

First create data structures that will hold the midi clip

(define ht (make-hash-table))
(define events '())

Now a helper function for holding note events from keyboard

(define (createevent time key vel dur)
  (let ((ht (make-hash-table)))
    (set! (ht 'time) time)
    (set! (ht 'key) key)
    (set! (ht 'vel) vel)
    (set! (ht 'dur) dur)    
    ht))

Midi notes are sent from a midi keyboard in pairs of note-on, note-off messages, so to determine the duration of the note we need to keep track of the time for those events. (metro-beat) gives us the current time. To keep track of this a hash table is used, ht. After a note-on event we store this in ht, when note-off comes we calculate the duration and create an event (which is a hash table again) using createevent function and store those hash tables in events list.

(define (recorder data)  
    (let* ((opcode (first data))
        (channel (second data))
        (keyn (third data))
        (velocity (fourth data))
        (curtime (metro-beat)))
    (if (= opcode mm:on)
        (begin 
            (mp:on :key keyn :vel velocity :chan channel)
            (set! (ht keyn) (list curtime velocity)))
    (if (= opcode mm:off)
        (let ((dur (- curtime (first (ht keyn)))))
            (let ((event (createevent (first (ht keyn)) keyn (second (ht keyn)) dur)))
                (set! events (cons event events)))                        
                (set! (ht keyn) #f)
                (mp:off :key keyn :chan channel))))))

Now start receiving events

(mp:receive recorder)

When you are done, reverse the list, this is necessary since the list is built by consing

(define ready_to_play (reverse events))

To play back correctly we need duration for the notes and the time to wait between notes. the function below calculates wait time and returns a new list.

In this case the wait time is stored in key 'time

(define (tdiff times)
  (if (eq? (length times) 1) (list (createevent 0 ((car times) 'key) ((car times) 'vel) ((car times) 'dur)))
      (cons (createevent (- ((second times) 'time) ((first times) 'time))

Function for playing back

(define (player clip)
  (process for e in clip
           do
           (print (/ (e 'vel) 127))
           (mp:midi :key (e 'key) :dur (e 'dur) :amp (/ (e 'vel) 127))
           (wait (e 'time))))

Start playback

(sprout (player (tdiff x)))
